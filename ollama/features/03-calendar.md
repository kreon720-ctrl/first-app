# 팀 캘린더 · 일정 관리

## 캘린더 화면 구성

팀 메인 화면(`/teams/[팀ID]`)은 캘린더 영역과 채팅 영역으로 구성됩니다.

- **데스크탑(1024px 이상)**: 좌측 캘린더 + 우측 채팅 분할 화면. 날짜를 클릭하면 우측 채팅도 그 날짜로 즉시 이동.
- **모바일(640px 미만)**: 상단 [캘린더] / [채팅] 탭으로 나눠지며 전환해 사용.

## 일정 보기 (모든 팀원 가능)

### 월간 / 주간 / 일간 뷰
- 캘린더 상단의 **[월] [주] [일]** 탭으로 뷰를 전환할 수 있습니다.
- 기본 뷰는 **월간(月)**입니다.
- `<` `>` 화살표로 이전 달·다음 달로 이동할 수 있습니다.

### 일정 표시
- 각 날짜 셀에 그 날의 일정 제목이 카드 형태로 나타납니다.
- 일정 카드를 클릭하면 상세 팝업이 열려 제목·설명·시작 일시·종료 일시를 볼 수 있습니다.
- 오늘 날짜는 시각적으로 강조됩니다.

### 빈 상태
해당 월에 일정이 없으면 "등록된 일정이 없습니다" 안내 문구가 표시됩니다.

## 일정 추가 (팀 구성원 누구나 가능)

1. 팀 메인 화면에서 `[+ 일정 추가]` 버튼을 누르거나 특정 날짜 셀을 클릭합니다.
2. 일정 생성 화면(`/teams/[팀ID]/schedules/new`) 또는 인라인 폼에서 아래 값을 입력합니다.
   - **제목**(필수, 최대 200자)
   - **설명**(선택)
   - **시작 일시**(필수)
   - **종료 일시**(필수, 시작 일시보다 뒤)
3. `[저장]` 버튼을 누르면 일정이 만들어지고 팀 전체가 즉시 볼 수 있습니다.

### 유효성 규칙
- 제목이 비어 있으면 "제목은 필수입니다" 인라인 오류.
- 제목 200자 초과 시 오류.
- 종료 시각이 시작 시각보다 같거나 이전이면 "종료 시각은 시작 시각 이후여야 합니다" 오류.

### 권한
- **팀장(LEADER)·팀원(MEMBER) 모두** 일정을 만들 수 있습니다.
- 팀에 속하지 않은 사용자가 다른 팀의 일정을 추가하려 하면 "권한이 없습니다" 오류가 납니다.

## 일정 수정 (생성자 본인만)

1. 캘린더에서 수정하려는 일정을 클릭해 상세 팝업을 엽니다.
2. 상세 팝업에서 `[수정]` 버튼을 누르면 편집 모드로 전환됩니다(기존 값이 채워진 상태).
3. 원하는 필드를 변경한 뒤 `[저장]` 버튼을 누릅니다.

### 권한 경계
- **일정을 만든 사람 본인**만 수정할 수 있습니다.
- 다른 사람이 만든 일정에는 `[수정]` 버튼 자체가 표시되지 않습니다.
- 만약 API를 직접 호출해 수정을 시도해도 "일정 수정 권한이 없습니다" 오류(403)가 납니다.

## 일정 삭제 (생성자 본인만)

1. 캘린더에서 삭제하려는 일정을 클릭해 상세 팝업을 엽니다.
2. `[삭제]` 버튼을 누릅니다.
3. "정말 삭제하시겠습니까?" 확인 다이얼로그에서 승인합니다.
4. 캘린더가 자동 갱신되고 해당 일정이 사라집니다.

### 권한 경계
- **일정을 만든 사람 본인**만 삭제할 수 있습니다.
- 다른 사람이 만든 일정에는 `[삭제]` 버튼이 나타나지 않습니다.
- 팀장이라도 다른 팀원이 만든 일정은 삭제할 수 없습니다.

## 날짜·채팅 연동

- 캘린더에서 특정 날짜를 클릭하면 **채팅 영역이 그 날짜의 메시지로 자동 전환**됩니다.
- 모바일에서는 캘린더에서 날짜를 선택한 뒤 [채팅] 탭으로 이동하면 선택한 날짜 메시지를 볼 수 있습니다.
- 화면 크기를 바꿔도 선택한 날짜는 유지됩니다.

## 알림

일정 추가·수정·삭제에 대해 푸시 알림이나 이메일 알림을 보내는 기능은 현재 안내되어 있지 않습니다. 팀원들은 직접 앱을 확인해야 합니다.

## 자주 묻는 사례

- **Q**: 일정이 갑자기 사라졌어요.
  **A**: 일정을 만든 사람이 삭제했을 가능성이 있습니다. 채팅에서 해당 팀원에게 확인해 보세요.

- **Q**: 내가 만든 일정인데 수정 버튼이 안 보여요.
  **A**: 로그인한 계정과 일정 생성자 계정이 다른지 확인해 보세요. 같은 이메일이라도 계정이 두 개면 다른 사용자로 인식됩니다.

- **Q**: 다른 팀 일정도 함께 볼 수 있나요?
  **A**: 현재는 팀별로 따로 봐야 합니다. 여러 팀의 일정을 한 화면에 모아서 보는 기능은 없습니다.
