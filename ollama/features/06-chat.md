# 팀 채팅 (일반 / 업무보고 / 공지사항)

## 개요

TEAM WORKS의 채팅은 목적에 따라 **세 가지 타입**으로 나뉩니다.

| 타입 | 누가 보내나 | 누가 보나 |
|------|------------|----------|
| 일반 대화 (NORMAL) | 팀 구성원 누구나 | 팀 전체 |
| 업무보고 (WORK_PERFORMANCE) | 팀 구성원 누구나 | 기본 팀장만. 팀장이 허용한 팀원 추가 |
| 공지사항 (Notice) | 팀 구성원 누구나 | 팀 전체 (채팅 상단 고정) |

## 공통 동작

### 날짜별 정리
- 채팅 메시지는 **작성 날짜(KST 기준)별로 자동 그룹핑**됩니다.
- 캘린더에서 특정 날짜를 클릭하면 채팅 영역이 자동으로 그 날짜로 이동합니다.
- "지난주 화요일에 뭐라고 했더라?"를 스크롤 없이 바로 찾을 수 있습니다.

### 갱신 방식 (폴링)
- 채팅은 **완전 실시간이 아닙니다**. 3~5초마다 자동으로 새 메시지를 받아옵니다.
- 본인이 보낸 메시지는 즉시 보입니다.
- 다른 팀원이 보낸 메시지는 최대 5초 정도 지연되어 나타날 수 있습니다.
- 이 방식은 "폴링"이라 부르며, 사용자는 따로 새로고침할 필요가 없습니다.

### 길이 제한
- 한 메시지의 최대 길이는 **2,000자**입니다.
- 빈 메시지는 전송할 수 없습니다.

## 일반 대화 전송

1. 팀 메인 화면의 채팅 영역 하단 입력창에 메시지를 씁니다.
2. `[전송]` 버튼을 누르거나 Enter 키를 입력합니다.
3. 메시지가 즉시 채팅 목록에 나타나고 다른 팀원에게도 수 초 내에 표시됩니다.

일반 대화는 특별한 분리 없이 모든 팀원이 자유롭게 주고받을 수 있습니다.

## 업무보고 전송 (WORK_PERFORMANCE)

팀원이 팀장에게 **공식적인 보고**를 올릴 때 쓰는 채팅 타입입니다.

1. 채팅 영역에서 `[업무보고]` 버튼을 누릅니다. 입력 모드가 업무보고 모드로 전환됩니다.
2. 보고 내용을 입력합니다.
3. `[전송]`을 누르면 메시지가 전송되고, 일반 대화와 시각적으로 구분되어 표시됩니다(다른 배경색 또는 뱃지).

### 가시성 규칙
- **팀장(LEADER)은 항상 모든 업무보고를 볼 수 있습니다.**
- **팀원(MEMBER)은 기본적으로 다른 사람의 업무보고를 볼 수 없습니다.**
- 팀장이 특정 팀원에게 **업무보고 열람 권한**을 부여하면, 그 팀원도 업무보고를 볼 수 있게 됩니다.
- 이 구조 덕분에 민감한 보고 내용(개인 평가 등)이 동료에게 노출되지 않습니다.

## 업무보고 열람 권한 관리 (팀장 전용)

1. 팀 설정에서 `[업무보고 열람 권한 관리]` 메뉴로 이동합니다.
2. 팀원 목록에서 열람을 **허용할 사람을 체크**합니다.
3. `[저장]`을 누릅니다.

### 특수 규칙
- **허용 목록을 빈 상태로 저장하면 전체 공개**가 됩니다. 이 경우 모든 팀원이 모든 업무보고를 볼 수 있게 됩니다.
- 특정 팀원만 선택해 저장하면 그 사람들만 열람 가능, 나머지 팀원은 차단됩니다.
- 변경은 즉시 반영됩니다(다음 채팅 조회부터).

### 권한 경계
- 팀원이 권한 설정을 시도하면 "팀장만 권한을 설정할 수 있습니다" 오류가 납니다.

## 공지사항 작성 · 삭제

공지사항은 중요한 알림을 **채팅 상단에 고정**해 모든 팀원에게 눈에 띄게 전달하는 기능입니다.

### 작성
1. 채팅 화면에서 `[공지 작성]` 버튼을 누릅니다.
2. 공지 내용을 입력합니다(최대 2,000자).
3. `[등록]`을 누르면 채팅 상단에 고정 공지로 표시됩니다.
4. **팀 구성원 누구나** 공지를 올릴 수 있습니다.

### 삭제 — 예외적인 권한 규칙
- **작성자 본인** 또는 **팀장(LEADER)**이 삭제할 수 있습니다.
- 다른 기능(일정·프로젝트·포스트잇)에서는 팀장이 다른 사람 것을 지울 수 없지만, **공지사항만은 팀장이 삭제 가능**합니다.
- 이유: 오래되었거나 부적절한 공지가 계속 고정되어 있지 않도록 팀장이 정리할 수 있게 하기 위함입니다.
- 작성자·팀장이 아닌 사람이 삭제를 시도하면 "권한이 없습니다" 오류(403)가 납니다.

## 자주 묻는 사례

- **Q**: 채팅이 5초 정도 늦게 갱신되는데 버그 아닌가요?
  **A**: 정상 동작이에요. TEAM WORKS는 3~5초 간격으로 자동 갱신하는 방식이라 약간의 지연이 있을 수 있어요. 본인이 보낸 건 바로 보입니다.

- **Q**: 내 업무보고를 동료가 보는 것 같아 걱정돼요.
  **A**: 기본은 팀장만 보는 구조예요. 팀장이 열람 권한을 명시적으로 부여한 팀원만 볼 수 있어요. 팀장에게 현재 권한 설정이 어떤지 확인해 달라고 부탁해 보세요.

- **Q**: 공지사항을 팀장이 지웠어요. 작성자에게 알림이 가나요?
  **A**: 삭제 시 별도 알림은 없어요. 공지가 사라진 것으로 확인할 수 있습니다.

- **Q**: 메시지를 수정할 수 있나요?
  **A**: 현재 전송한 메시지를 수정하는 기능은 안내되어 있지 않아요. 새 메시지로 정정 내용을 올리셔야 해요.

- **Q**: 메시지를 삭제할 수 있나요?
  **A**: 개별 채팅 메시지 삭제 기능은 안내되어 있지 않아요. 공지사항만 삭제가 가능합니다.

- **Q**: 파일이나 이미지를 보낼 수 있나요?
  **A**: 현재 텍스트 메시지만 지원되며, 파일 첨부 기능은 안내되어 있지 않아요.
