# 회원가입 · 로그인

## 개요

TEAM WORKS를 이용하려면 이메일 계정으로 회원가입한 뒤 로그인해야 합니다. 모든 기능(팀 생성, 일정 조회, 채팅 등)은 로그인한 사용자에게만 열려 있습니다.

## 회원가입 절차

1. 초기 화면에서 `[회원가입]` 버튼을 누릅니다.
2. 아래 정보를 입력합니다.
   - **이메일**: 올바른 이메일 형식(예: `name@example.com`). 중복 불가.
   - **이름**: 최대 50자.
   - **비밀번호**: 보안 강도 기준 충족 필요.
3. `[가입하기]`를 누르면 계정이 생성되고 로그인 토큰이 발급됩니다.

### 유효성 오류
- 이메일 형식 오류 → "올바른 이메일 형식이 아닙니다".
- 이미 가입된 이메일 → "이미 사용 중인 이메일입니다".
- 이름 50자 초과 → "이름은 최대 50자까지 입력 가능합니다".

## 로그인 절차

1. 이메일과 비밀번호 입력.
2. `[로그인]` 버튼 클릭.
3. 인증 성공 시 팀 목록 홈(`/`)으로 이동합니다.

### 로그인 실패 사례
- 잘못된 이메일/비밀번호 → "이메일 또는 비밀번호가 올바르지 않습니다".
- 가입되지 않은 이메일 → 동일한 오류 메시지(보안상 세부 원인 비공개).

## 세션 유지

- 인증은 JWT(Access Token + Refresh Token) 방식입니다.
- 브라우저를 닫아도 일정 시간 동안 세션이 유지됩니다.
- 세션 만료 시 자동으로 로그인 화면으로 이동하며 다시 로그인해야 합니다.

## 로그아웃

- 상단 메뉴의 `[로그아웃]` 또는 프로필 영역에서 로그아웃 버튼을 누릅니다.
- 로그아웃 시 세션이 종료되고 로그인 화면으로 돌아갑니다.

## 비밀번호 보안

- 비밀번호는 서버에 **bcrypt 해시**로 저장되며 원문은 저장되지 않습니다.
- 분실한 비밀번호 찾기/재설정 기능은 현재 안내되어 있지 않습니다.

## 자주 묻는 사례

- **Q**: 회원가입이 갑자기 안 돼요.
  **A**: 이메일 형식이 맞는지, 이미 쓴 이메일은 아닌지 먼저 확인해 보세요. 반복되면 브라우저 캐시를 지우고 다시 시도하거나 관리자에게 문의해 주세요.

- **Q**: 같은 이메일로 두 계정을 만들 수 있나요?
  **A**: 안 됩니다. 이메일은 고유해야 하며 중복 가입 시 오류가 납니다.

- **Q**: 비밀번호를 잊어버렸어요.
  **A**: 현재 비밀번호 재설정 기능은 안내되어 있지 않아요. 관리자에게 문의해 주세요.

- **Q**: 로그아웃하지 않으면 계속 로그인 상태인가요?
  **A**: 일정 시간 동안은 자동 로그인 상태가 유지됩니다. 토큰이 만료되면 다시 로그인해야 해요.

- **Q**: 다른 기기에서 동시에 로그인할 수 있나요?
  **A**: 특별한 제약은 안내되어 있지 않아요. 다만 보안을 위해 공용 PC에서는 반드시 로그아웃해 주세요.
