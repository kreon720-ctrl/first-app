# 팀 관리 (생성 · 탐색 · 가입 · 승인)

## 역할 구조

TEAM WORKS에는 팀마다 두 가지 역할이 있습니다.

- **팀장(LEADER)**: 팀을 만든 사람. 팀당 1명. 자동 지정.
- **팀원(MEMBER)**: 팀장의 승인으로 팀에 합류한 사용자.

## 팀 만들기

1. 로그인 후 팀 목록 홈 화면(`/`)으로 이동합니다.
2. 화면 우측 하단의 `[+ 팀 생성]` 버튼(또는 빈 상태의 CTA 버튼)을 누릅니다.
3. 팀 생성 화면(`/teams/new`)에서 **팀 이름**(1~100자)을 입력합니다.
4. `[생성]` 버튼을 누르면 팀이 만들어지고 본인이 자동으로 팀장(LEADER)으로 등록됩니다.
5. 생성된 팀의 메인 화면(`/teams/[팀ID]`)으로 이동합니다.

### 규칙
- 팀 이름은 필수이며 빈 값이면 버튼이 비활성화됩니다.
- 100자를 넘으면 "팀 이름은 최대 100자까지 입력 가능합니다" 오류.
- **팀 생성과 동시에 팀장이 됩니다.** 별도 설정이 필요 없습니다.

### 공개 팀 설정

팀을 만들 때 또는 이후에 팀을 **공개(isPublic)** 상태로 두면 다른 사용자들이 공개 팀 목록에서 팀을 찾아 가입 신청을 할 수 있습니다. 비공개면 공개 목록에 노출되지 않습니다.

## 공개 팀 탐색 (가입을 원할 때)

TEAM WORKS에는 **초대 기능이 없습니다**. 팀에 들어가려면 공개된 팀을 직접 찾아 신청해야 합니다.

1. 팀 목록 홈에서 `[팀 탐색]` 버튼을 누릅니다.
2. 공개 팀 목록 화면(`/teams/explore`)에서 팀 카드들을 둘러봅니다. 각 카드에는 팀명, 팀장 이름, 구성원 수가 표시됩니다.
3. 원하는 팀의 `[가입 신청]` 버튼을 누릅니다.
4. 신청이 전송되면 "[팀명]에 가입 신청을 완료했습니다. 팀장의 승인을 기다려주세요" 안내가 나타나고 버튼이 "신청 완료" 상태로 비활성화됩니다.

### 중복 신청 규칙
- 이미 그 팀의 구성원인 경우 → "이미 해당 팀의 구성원입니다" 오류.
- 이미 대기 중(PENDING) 신청이 있는 경우 → "이미 가입 신청이 진행 중입니다" 오류.

### 가입 신청 취소
현재 신청자 본인이 신청을 취소하는 기능은 따로 안내되어 있지 않습니다. 필요하다면 팀장에게 직접 거절 처리를 요청하세요.

## 가입 신청 승인/거절 (팀장 전용)

### 진입 경로

팀장은 상단 네비게이션 또는 팀 메인 화면에서 `[나의 할 일]` 버튼을 누르면 '나의 할 일' 화면(`/me/tasks`)으로 이동합니다.

### 처리 방법

1. '나의 할 일' 화면에 내가 팀장인 **모든 팀의 대기 중(PENDING) 가입 신청**이 리스트로 표시됩니다.
2. 각 신청 카드에 신청자 이름·이메일·신청 일시·대상 팀명이 나옵니다.
3. `[승인]` 또는 `[거절]` 버튼을 누릅니다.

- **승인**: 신청자가 즉시 해당 팀의 팀원(MEMBER)으로 등록됩니다. 신청 항목은 목록에서 제거됩니다.
- **거절**: 팀 합류 없이 처리 종료. 신청자에게 별도 알림은 없습니다(화면에서 버튼이 비활성화됩니다).

### 권한 경계
- 팀원(MEMBER)이 승인/거절 API를 직접 호출하면 "팀장만 가입 신청을 처리할 수 있습니다" 오류가 납니다.
- 이미 승인/거절된 신청을 다시 처리하려 하면 "이미 처리된 가입 신청입니다" 오류.

## 여러 팀 관리

- 한 사용자가 여러 팀에 속할 수 있습니다(각기 팀장일 수도, 팀원일 수도 있습니다).
- '나의 할 일' 화면은 **내가 팀장인 모든 팀의 신청**을 한 번에 모아 보여줍니다.
- 팀 간 일정·채팅은 완전히 격리됩니다. 한 팀 구성원은 다른 팀의 내용을 볼 수 없습니다.

## 팀에서 나가기 / 팀장 변경

- **팀 탈퇴**: 현재 안내된 기능이 없습니다.
- **팀장 양도**: 현재 안내된 기능이 없습니다. 팀을 만든 사람이 팀장으로 유지됩니다.
- **팀 삭제**: 현재 안내된 기능이 없습니다.

## 관련 화면

| 화면 | 경로 | 접근 |
|------|------|------|
| 팀 목록 홈 | `/` | 로그인 사용자 |
| 팀 생성 | `/teams/new` | 로그인 사용자 |
| 공개 팀 탐색 | `/teams/explore` | 로그인 사용자 |
| 나의 할 일 | `/me/tasks` | 팀장만 의미 있음 |
| 팀 메인 | `/teams/[팀ID]` | 해당 팀 구성원 |

## 자주 묻는 사례

- **Q**: 이메일로 팀원을 초대할 수 있나요?
  **A**: 초대 기능은 없어요. 본인이 공개 팀을 찾아 직접 가입 신청해야 합니다. 팀장이 공개(isPublic) 상태로 바꿔두면 신청이 가능합니다.

- **Q**: 팀원을 강제로 내보낼 수 있나요?
  **A**: 현재 팀원 제거 기능은 안내되어 있지 않아요.

- **Q**: 가입 신청을 잘못 보냈는데 취소하려면요?
  **A**: 신청 취소 기능은 안내되어 있지 않아요. 해당 팀장에게 거절 처리를 요청하시면 됩니다.
