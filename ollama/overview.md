# TEAM WORKS 한 눈에 보기

## TEAM WORKS는 무엇인가요

TEAM WORKS는 팀 단위로 **일정과 채팅을 한 화면에서** 관리하는 협업 도구입니다. 메신저로는 일정의 맥락을 잃어버리고, 캘린더 도구로는 대화가 안 되는 문제를 해결합니다. 한 팀의 캘린더와 채팅, 공지, 업무보고, 포스트잇이 모두 같은 화면 안에 있어 "그 회의 어디서 정했더라" 같은 검색을 할 필요가 없습니다.

## 누구를 위한 도구인가요

- **소규모 팀(3~30명)**의 협업 — 학교 동아리, 스타트업 팀, 사내 TF, 프리랜서 협업 그룹
- 별도 채팅 앱 + 캘린더 + 문서 도구를 쓰는 게 부담스러운 팀
- "오늘 누가 뭐 하는지" + "다음 주에 뭐가 있는지"가 한 화면에 보였으면 하는 팀

## 핵심 컨셉 5가지

1. **팀 단위 격리** — 한 팀의 일정·채팅·포스트잇·프로젝트는 그 팀 안에서만 보입니다. 다른 팀 정보는 절대 노출되지 않습니다.
2. **생성자 기반 권한** — 내가 만든 것만 내가 수정/삭제합니다. 팀장이라도 다른 사람의 일정을 임의로 고치지 못합니다.
3. **공개 팀 + 가입 신청** — 다른 팀에 들어가려면 그 팀 공개 목록에서 가입 신청 → 팀장 승인 절차를 거칩니다.
4. **업무보고 ≠ 일반 채팅** — 평가성 보고는 별도 메시지 타입(WORK_PERFORMANCE)으로, 기본은 팀장만 봅니다. 동료가 훔쳐볼 수 없는 안전한 구조.
5. **AI 비서 "찰떡"** — 사용법 안내(검색)와 일정 등록 자동화(에이전트) 두 모드 지원.

## 주요 기능 목록

| 기능 | 설명 |
|------|------|
| 회원가입 / 로그인 | 이메일·비밀번호 기반. JWT 인증 |
| 팀 만들기 | 팀명·설명 입력, 만든 사람이 자동으로 팀장 |
| 팀 탐색 / 가입 신청 | 공개 팀 목록 → 가입 신청 → 팀장 승인 |
| 나의 할 일 | 내가 팀장인 모든 팀의 가입 신청 모아보기 |
| 팀 캘린더 | 월·주·일 뷰. 일정 추가·수정·삭제 |
| 포스트잇 | 날짜에 붙이는 메모. 5가지 색상 |
| 팀 채팅 | 일반 메시지(NORMAL) — 팀원 모두 조회 |
| 업무보고 | 별도 타입(WORK_PERFORMANCE). 팀장만 / 권한받은 팀원만 조회 |
| 공지사항 | 팀 채팅 상단 고정 알림 |
| 프로젝트 | 간트차트 형태. 단계 → 프로젝트 일정 → 세부 일정 3단계 |
| AI 비서 (찰떡) | 사용법 Q&A + 일정 조회·등록 자동화 |

## 사용자 역할 두 가지

**팀장 (LEADER)**
- 팀을 만든 사람. 팀당 1명.
- 가입 신청 승인/거절 가능
- 업무보고 열람 권한을 다른 팀원에게 부여 가능
- 자신이 만든 콘텐츠만 수정/삭제 가능 (팀장이라도 남의 것 못 건드림)
- 단, 다른 팀원이 작성한 공지사항은 팀장 권한으로 삭제 가능

**팀원 (MEMBER)**
- 팀에 가입된 일반 사용자
- 팀 일정·채팅·포스트잇·프로젝트·공지 모두 작성 가능
- 자신이 만든 콘텐츠만 수정/삭제 가능
- 업무보고는 팀장이 권한 부여한 경우에만 다른 팀원의 보고를 열람 가능

## 한 사용자가 여러 팀에 속할 수 있나요

네. 한 계정으로 여러 팀의 LEADER가 될 수도 있고, 어떤 팀에서는 LEADER, 다른 팀에서는 MEMBER일 수도 있습니다. "나의 할 일" 메뉴에서 내가 팀장인 모든 팀의 PENDING 가입 신청을 한 번에 볼 수 있습니다.
